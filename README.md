# MaxAlert 🚨🐥

일정을 자꾸 까먹는 사람을 위한 **강제성 있는** 데스크톱 알림 앱 (Windows / macOS).
마스코트는 병아리 경찰관 **맥스 경관(Officer Max)** — 사이렌이 울리면 맥스가 순찰을 돕니다.

## 핵심 기능

| 기능 | 설명 |
|---|---|
| 🗒️ 포스트잇 위젯 | 화면 우측에 색상별 포스트잇으로 오늘의 일정 상시 표시 |
| 💣 폭탄 시각화 | 마감 1시간 전 폭탄 등장 → 30분 전 도화선 점화 → 10분 전 흔들림 → 3분 전 패닉 → 시간 초과 시 💥 폭발 |
| 🚨 코드블루 사이렌 | 일정 **3분 전** 모든 모니터를 전체 화면으로 점거 + 사이렌 사운드. **완료 체크 또는 미루기 전까지 절대 꺼지지 않음** |
| 🐥 맥스 경관 | 사이렌이 뜨면 병아리 경찰관 맥스가 왼쪽에서 걸어 들어와 노려보다가 오른쪽으로 행진하며 순찰 (AI 생성 3D 영상 + 실시간 크로마키, 무한 루프) |
| 🗂️ 서랍식 포스트잇 | 3시간 이상 남은 일정은 끄트머리만 삐죽 → 3시간/1시간 전 단계적으로 나옴 → 30분 전 완전히 나오며 폭탄 점화 |
| ⚡ 포인트 & 레벨 | 완료 +10, 정시 보너스 +10, 미루기 -3, 전체 완료 +50, 뱃지 +25. 누적 포인트로 10단계 칭호 (산만한 금붕어 → 타임 마스터) |
| 🛍️ 포인트 상점 | 모은 ⚡로 포스트잇 테마 해금: 네온(150) · 크라프트(200) · 미드나잇(300) |
| 🏅 뱃지 8종 | 첫 걸음, 얼리버드, 퍼펙트 데이, 정면돌파, 작심삼일 극복, 일주일의 기적, 사이렌 슬레이어, 백전노장 |
| 📝 아침 투두리스트 | 앱 실행 시(부팅 자동 시작 지원) 시간+일정 입력 대시보드가 열림 |
| 🔗 노션 연동 | 노션 DB의 오늘 일정을 1분마다 자동 동기화, 완료 체크 양방향 반영 |
| 🏆 보상 | 오늘의 할 일 전부 완료 시 폭죽 + 칭찬 + 🔥 연속 달성 스트릭 |

## 실행

```bash
npm install
npm start
```

앱은 트레이(작업 표시줄 알림 영역)에 상주합니다. 창을 닫아도 백그라운드에서 계속 감시하며, 종료는 트레이 아이콘 우클릭 → 종료.

## 노션 연동 방법

1. [notion.so/my-integrations](https://www.notion.so/my-integrations)에서 통합(Integration) 생성 → **토큰** 복사
2. 일정 데이터베이스 페이지에서 `⋯ → 연결(Connections) → 통합 선택`으로 권한 부여
3. DB URL에서 32자리 **데이터베이스 ID** 복사 (URL 전체를 붙여넣어도 자동 추출됨)
4. 앱 대시보드 → `🔗 노션 연동`에 입력 후 저장

DB 속성 요건 (이름 무관, 타입으로 자동 감지):
- **제목**(title) — 할 일 이름
- **날짜**(date) — 시간까지 포함해야 사이렌 발동 (날짜만 있으면 종일 일정)
- **체크박스**(checkbox) — 완료 여부 (선택, 양방향 동기화)

## 동작 규칙

- 시간이 지정된 일정은 **3분 전부터** 사이렌이 울립니다.
- 사이렌은 `✅ 완료` 또는 `⏳ 미루기(5분/15분/1시간)`를 누를 때까지 꺼지지 않고, 계속 최상위 화면을 점거합니다.
- 이미 지난 시간으로 등록/동기화된 일정은 즉시 사이렌을 울리지 않습니다 (포스트잇에는 💥 표시).
- 데이터는 로컬(`%APPDATA%/byeadhd` 또는 `~/Library/Application Support/byeadhd`)에 저장됩니다.

## 알려진 한계

- 사이렌의 "강제성"은 전체 화면 점거 + 0.7초마다 포커스 재탈환 방식입니다. OS 보안상 키보드 입력(Alt+Tab 등)을 완전히 차단할 수는 없지만, 도망가도 즉시 다시 덮습니다.
- 배포용 패키징(electron-builder)은 아직 미설정 — 현재는 `npm start`로 실행.
